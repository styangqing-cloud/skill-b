# employee-interview-generator

互联网公司员工访谈提纲生成器

---

## 📋 技能信息

| 项目 | 说明 |
|------|------|
| **技能名称** | employee-interview-generator |
| **版本** | 3.0.1 |
| **功能** | 互联网公司员工访谈提纲生成器 |

---

## ✨ 核心功能

- 📝 **支持 9 种访谈类型**：入职/绩效/离职/晋升/职业发展/团队反馈/挽留/转岗/组织诊断
- 📁 **支持文件上传分析**：Excel/PDF/PPTX/Word/TXT
- 🔗 **支持网页链接分析**
- 📝 **腾讯文档集成**：一键生成和同步访谈提纲

---

## 🎯 专业方法论支持

| 方法论 | 适用场景 |
|-------|---------|
| 韦斯伯德六个盒子模型 | 组织诊断访谈 |
| 推拉力理论 | 离职访谈 |
| 冰山模型 | 招聘需求访谈 |
| 吉尔伯特 BEM 模型 | 员工发展访谈 |
| 职业锚理论 + GROW 模型 | 职业发展访谈 |
| 杨三角理论 | 组织能力诊断 |

---

## 💼 使用场景

- HR 访谈提纲生成
- 组织健康度诊断
- 离职原因分析
- 员工发展规划
- 晋升面试评估
- 团队反馈收集

```bash
# 克隆仓库
git clone https://github.com/styangqing-cloud/skill-b.git

# 复制技能到 OpenClaw 技能目录
cp -r skill-b/employee-interview-generator /root/.openclaw/workspace/skills/
```

## 🚀 快速开始

### 员工访谈

```
帮我生成一份离职访谈提纲，员工是腾讯 T12 产品经理
```

### 招聘面试

```
帮我生成一份招聘面试提纲，职位是高级后端工程师
```

```
分析这份简历，生成员工访谈提纲
```
