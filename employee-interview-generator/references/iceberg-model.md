# 招聘需求访谈指南 - 冰山模型

本指南为使用**冰山模型**（Iceberg Model）进行招聘需求访谈提供详细指导。

---

## 📊 理论介绍

冰山模型由心理学家 David McClelland 于 1973 年提出，将人的素质分为水面以上（可见）和水面以下（隐藏）两部分：

```
                    ┌─────────────────┐
                    │   水面以上      │
                    │   (可见 20%)    │
                    ├─────────────────┤
                    │   知识          │
                    │   技能          │
                    ├═════════════════┤ ← 水面线
                    │   社会角色      │
        水面以下    │   自我认知      │
        (隐藏 80%)  │   特质          │
                    │   动机          │
                    └─────────────────┘
```

**核心逻辑：**
- **水面以上（知识、技能）**：容易评估和培养
- **水面以下（角色、认知、特质、动机）**：难以改变，决定长期绩效

---

## 🎯 六个层次详解

### 1️⃣ 知识（Knowledge）

**定义：** 岗位所需的专业知识和信息

**评估问题：**
- 这个岗位需要什么专业知识？
- 哪些知识是必须的？哪些是加分项？
- 知识缺口可以通过培训弥补吗？

**访谈问题示例：**

```
主问题：这个岗位需要什么专业知识？

追问提示：
- 哪些知识是入职就必须具备的？
- 哪些知识可以在工作中学习？
- 如果候选人知识有欠缺，多长时间内可以弥补？
```

---

### 2️⃣ 技能（Skills）

**定义：** 完成工作所需的技术和能力

**评估问题：**
- 完成工作需要什么硬技能？
- 需要什么软技能？
- 哪些技能可以培训？哪些必须现有？

**访谈问题示例：**

```
主问题：完成这个岗位的工作，最关键的 3 个技能是什么？

追问提示：
- 这些技能中，哪些是必须入职就具备的？
- 哪些技能可以通过培训获得？
- 如果技能有欠缺，对工作的影响有多大？
```

---

### 3️⃣ 社会角色（Social Role）

**定义：** 个人在社会/组织中的角色定位和价值观

**评估问题：**
- 这个岗位在团队中扮演什么角色？
- 需要什么样的价值观？
- 如何与团队文化匹配？

**访谈问题示例：**

```
主问题：这个岗位在团队中扮演什么角色？

追问提示：
- 团队期望这个人带来什么价值？
- 这个岗位需要什么样的工作风格？
- 如何与现有团队文化匹配？
```

---

### 4️⃣ 自我认知（Self-Image）

**定义：** 个人对自己的认知和定位

**评估问题：**
- 候选人如何定位自己？
- 与岗位期望是否一致？
- 自我认知是否清晰？

**访谈问题示例：**

```
主问题：你希望候选人如何定位自己在团队中的角色？

追问提示：
- 是执行者、协调者、还是创新者？
- 希望候选人有多大的自主性？
- 如何平衡自信与谦逊？
```

---

### 5️⃣ 特质（Traits）

**定义：** 个人的性格特征和行为倾向

**评估问题：**
- 这个岗位需要什么性格特质？
- 哪些特质是成功的关键？
- 如何评估这些特质？

**访谈问题示例：**

```
主问题：这个岗位最需要什么样的性格特质？

追问提示：
- 是外向还是内向更适合？
- 需要什么样的抗压能力？
- 需要什么样的决策风格？
```

---

### 6️⃣ 动机（Motives）

**定义：** 驱动个人行为的内在动力

**评估问题：**
- 什么驱动这个岗位的成功？
- 候选人的动机与岗位匹配吗？
- 如何激励这个岗位的人？

**访谈问题示例：**

```
主问题：什么驱动这个岗位的人成功？

追问提示：
- 是成就动机、权力动机、还是亲和动机？
- 什么能激励这个岗位的人？
- 候选人的动机与岗位匹配吗？
```

## 🔍 岗位画像对齐流程

### 步骤 1：明确岗位目标

**问题：**
- 这个岗位的核心目标是什么？
- 成功的标准是什么？
- 这个岗位解决什么问题？

### 步骤 2：分析六个层次

针对每个层次，问：
- 这个层次的要求是什么？
- 哪些是必须的？哪些是加分的？
- 如何评估这个层次？

### 步骤 3：优先级排序

将六个层次的要求按重要性排序：
- **必须满足**（底线）
- **应该满足**（重要）
- **可以培养**（可接受缺口）

### 步骤 4：制定评估策略

针对每个层次，制定评估方法：
- 知识/技能 → 笔试/实操
- 社会角色/自我认知 → 行为面试
- 特质/动机 → 心理测评/深度面试

## 💬 招聘需求访谈完整提纲

### 一、岗位背景（10 分钟）

**主问题 1：** 这个岗位的核心目标是什么？

**追问提示：**
- 这个岗位解决什么问题？
- 成功的标准是什么？
- 这个岗位在组织中的位置？

**主问题 2：** 为什么需要招聘这个岗位？

**追问提示：**
- 是新增岗位还是替换？
- 如果是替换，前任为什么离开？
- 如果是新增，业务背景是什么？

---

### 二、冰山以上层次（20 分钟）

#### 知识层面

**主问题：** 这个岗位需要什么专业知识？

**追问提示：**

- 如何评估知识水平？

- 如何评估技能水平？

---

### 四、优先级与评估（15 分钟）

**主问题 1：** 在这六个层次中，哪些是必须满足的底线？

**追问提示：**
- 如果某个层次不满足，会直接淘汰吗？
- 哪些层次可以适当放宽？
- 哪些层次可以入职后培养？

**主问题 2：** 如何评估候选人的匹配度？

**追问提示：**
- 每个层次用什么方法评估？
- 评估的权重如何分配？
- 谁是最终决策者？

---

### 五、总结与确认（5 分钟）

**参考话术：**
> "我来总结一下这个岗位的画像... [复述要点]"
> 
> "其中必须满足的底线是：[列出底线]"
> 
> "可以培养或放宽的是：[列出可培养项]"
> 
> "评估方法是：[列出评估策略]"
> 
> "还有什么需要补充或调整的吗？"

## ⚠️ 注意事项

### 常见误区

| 误区 | 表现 | 避免方法 |
|-----|------|---------|
| **重上轻下** | 只关注知识技能，忽视冰山以下 | 强制要求分析六个层次 |
| **完美主义** | 所有层次都要求"必须" | 区分必须/应该/加分 |
| **主观臆断** | 基于个人偏好而非岗位需求 | 基于岗位目标推导 |
| **忽视文化** | 忽视社会角色和文化匹配 | 明确团队文化期望 |

### 评估建议

| 层次 | 推荐评估方法 |
|-----|-------------|
| 知识 | 笔试、问答 |
| 技能 | 实操、案例分析 |
| 社会角色 | 行为面试、团队面试 |
| 自我认知 | 深度面试、360 评估 |
| 特质 | 心理测评、情境面试 |
| 动机 | 深度面试、职业锚测评 |
