# 职业发展访谈指南 - 职业锚理论 + GROW 模型

本指南为结合**职业锚理论**（Career Anchors）与**GROW 模型**进行职业发展访谈提供详细指导。

---

## 📊 理论介绍

### 职业锚理论（Career Anchors）

由美国麻省理工学院 Edgar Schein 教授提出，指个人在职业选择和发展中始终不会放弃的核心价值观和动机：

```
┌─────────────────────────────────────────────────────┐
│                  8 种职业锚类型                      │
├─────────────────────────────────────────────────────┤
│  1. 技术/职能型 (Technical/Functional)              │
│     - 追求专业技能的精进和应用                       │
├─────────────────────────────────────────────────────┤
│  2. 管理型 (General Managerial)                     │
│     - 追求承担管理责任和领导团队                     │
├─────────────────────────────────────────────────────┤
│  3. 自主/独立型 (Autonomy/Independence)             │
│     - 追求工作自主性和独立性                         │
├─────────────────────────────────────────────────────┤
│  4. 安全/稳定型 (Security/Stability)                │
│     - 追求工作稳定性和可预测性                       │
├─────────────────────────────────────────────────────┤
│  5. 创业/创造型 (Entrepreneurial Creativity)        │
│     - 追求创造新事物和挑战                           │
├─────────────────────────────────────────────────────┤
│  6. 服务/奉献型 (Service/Dedication)                │
│     - 追求帮助他人和社会贡献                         │
├─────────────────────────────────────────────────────┤
│  7. 纯挑战型 (Pure Challenge)                       │
│     - 追求解决难题和战胜强敌                         │
├─────────────────────────────────────────────────────┤
│  8. 生活方式型 (Lifestyle)                          │
│     - 追求工作与生活的平衡                           │
└─────────────────────────────────────────────────────┘
```

### GROW 模型

由 Sir John Whitmore 提出，是经典的教练式对话框架：

```
┌─────────────────────────────────────────────────────┐
│                  GROW 模型流程                       │
├─────────────────────────────────────────────────────┤
│  G - Goal（目标）                                   │
│     你想达成什么？                                  │
├─────────────────────────────────────────────────────┤
│  R - Reality（现状）                                │
│     你现在在哪里？                                  │
├─────────────────────────────────────────────────────┤
│  O - Options（选择）                                │
│     你有什么选择？                                  │
├─────────────────────────────────────────────────────┤
│  W - Will（意愿/行动）                              │
│     你将采取什么行动？                              │
└─────────────────────────────────────────────────────┘
```

## 🎯 8 种职业锚详解

### 1️⃣ 技术/职能型（Technical/Functional）

**核心特征：**
- 追求专业技能的精进
- 喜欢挑战性的专业工作
- 不喜欢管理工作

**访谈问题：**
```
- 你最喜欢工作中的哪部分？为什么？
- 你希望成为哪个领域的专家？
- 管理岗位对你有吸引力吗？为什么？
```

---

### 2️⃣ 管理型（General Managerial）

**核心特征：**
- 追求承担管理责任
- 喜欢领导和影响他人
- 追求综合性的工作

**访谈问题：**
```
- 你有带团队的经历吗？感受如何？
- 你希望管理多大的团队？
- 你认为什么是好的领导力？
```

---

### 3️⃣ 自主/独立型（Autonomy/Independence）

**核心特征：**
- 追求工作自主性
- 不喜欢被管控
- 喜欢独立工作

**访谈问题：**
```
- 你理想的工作方式是什么？
- 你希望有多大的自主权？
- 什么样的管理方式让你不舒服？
```

---

### 4️⃣ 安全/稳定型（Security/Stability）

**核心特征：**
- 追求工作稳定性
- 不喜欢风险和变化
- 重视福利和保障

**访谈问题：**
```
- 工作稳定性对你来说有多重要？
- 你如何看待职业风险？
- 什么能让你感到安心？
```

---

### 5️⃣ 创业/创造型（Entrepreneurial Creativity）

**核心特征：**
- 追求创造新事物
- 喜欢挑战和冒险
- 不喜欢按部就班

**访谈问题：**
```
- 你有过创业或从 0 到 1 的经历吗？
- 你理想的工作是什么样的？
- 什么能让你保持激情？
```

---

### 6️⃣ 服务/奉献型（Service/Dedication）

**核心特征：**
- 追求帮助他人
- 重视社会价值
- 喜欢有意义的工作

**访谈问题：**
```
- 工作对你来说意味着什么？
- 你希望自己的工作对他人有什么影响？
- 什么能让你感到满足？
```

---

### 7️⃣ 纯挑战型（Pure Challenge）

**核心特征：**
- 追求解决难题
- 喜欢竞争和战胜困难
- 不喜欢简单重复的工作

**访谈问题：**
```
- 你遇到过最有挑战的工作是什么？
- 什么能让你兴奋？
- 你如何面对困难？
```

---

### 8️⃣ 生活方式型（Lifestyle）

**核心特征：**
- 追求工作生活平衡
- 重视个人时间和家庭
- 不喜欢过度加班

**访谈问题：**
```
- 你理想的工作生活平衡是什么样的？
- 工作在你生活中的位置是什么？
- 什么对你来说最重要？
```

## 🔍 职业锚识别方法

### 方法 1：直接询问

```
主问题：回顾你的职业生涯，什么时候你感到最有成就感？

追问提示：
- 当时在做什么？
- 为什么让你有成就感？
- 这反映了你重视什么？
```

### 方法 2：职业选择分析

```
主问题：你过去做过哪些重要的职业选择？

追问提示：
- 当时有哪些选项？
- 为什么选择这个？
- 放弃的选项有什么吸引你的地方？
```

### 方法 3：未来愿景

```
主问题：想象 5 年后的理想工作状态，是什么样的？

追问提示：
- 工作内容是什么？
- 工作环境是什么样的？
- 工作与生活如何平衡？
```

## 💬 职业发展访谈完整提纲（GROW 模型框架）

### 一、Goal - 目标设定（20 分钟）

#### 职业锚探索

**主问题 1：** 回顾你的职业生涯，什么时候你感到最有成就感？

**追问提示：**
- 当时在做什么？
- 为什么让你有成就感？
- 这反映了你重视什么？

**主问题 2：** 你理想的工作状态是什么样的？

**追问提示：**
- 工作内容是什么？
- 工作环境是什么样的？
- 工作与生活如何平衡？

**主问题 3：** 5 年后，你希望自己在什么位置？

**追问提示：**
- 职位/职级是什么？
- 负责什么工作？
- 具备什么能力？

#### 目标澄清

**主问题：** 基于刚才的讨论，你的职业发展目标是什么？

**追问提示：**
- 短期（1 年）目标是什么？
- 中期（3 年）目标是什么？
- 长期（5 年）目标是什么？

---

### 二、Reality - 现状分析（20 分钟）

#### 当前状态

**主问题 1：** 你现在的工作状态怎么样？

**追问提示：**
- 工作内容符合预期吗？
- 工作环境满意吗？
- 工作生活平衡如何？

**主问题 2：** 你现在的核心能力是什么？

**追问提示：**
- 你的优势是什么？
- 哪些能力被充分利用？
- 哪些能力没有被发挥？

#### 差距分析

**主问题：** 现状和目标之间的差距是什么？

**追问提示：**
- 能力差距在哪里？
- 经验差距在哪里？
- 机会差距在哪里？

---

### 三、Options - 选择探索（15 分钟）

#### 发展路径

**主问题 1：** 达成目标，你有什么可能的路径？

**追问提示：**
- 内部发展有哪些机会？
- 外部机会有哪些？
- 还有什么其他选择？

#### 资源盘点

**主问题 2：** 你有什么资源可以利用？

**追问提示：**
- 公司内部资源有哪些？
- 外部人脉和资源有哪些？
- 还有什么可以争取的？

#### 障碍识别

**主问题：** 可能遇到什么障碍？

**追问提示：**
- 内部障碍是什么？
- 外部障碍是什么？
- 如何克服这些障碍？

---

### 四、Will - 行动计划（15 分钟）

#### 行动承诺

**主问题 1：** 接下来你将采取什么行动？

**追问提示：**
- 第一步做什么？
- 什么时候开始？
- 需要谁的支持？

#### 支持需求

**主问题 2：** 你需要什么支持？

**追问提示：**
- 公司可以提供什么支持？
- 上级可以提供什么支持？
- 还有什么需要的？

#### 跟进计划

**主问题：** 我们如何跟进进展？

**追问提示：**
- 多久跟进一次？
- 如何衡量进展？
- 如果进展不顺利怎么办？

---

### 五、总结与承诺（5 分钟）

**参考话术：**
> "我来总结一下今天的讨论... [复述要点]"
> 
> "你的职业锚是：[锚类型]"
> 
> "你的发展目标是：[目标]"
> 
> "接下来你将采取的行动：[行动项]"
> 
> "公司提供的支持：[支持项]"
> 
> "我们约定下次跟进时间：[时间]"

## ⚠️ 注意事项

### 职业锚识别原则

| 原则 | 说明 |
|-----|------|
| **自我发现** | 职业锚是员工自己发现的，而非被告诉 |
| **基于经历** | 基于真实职业经历，而非想象 |
| **动态稳定** | 职业锚相对稳定，但可能随经历微调 |
| **无好坏之分** | 8 种锚无优劣，只有匹配与否 |

### GROW 模型使用技巧

| 技巧 | 说明 |
|-----|------|
| **多问少说** | 80% 时间听，20% 时间说 |
| **开放问题** | 多用"什么"、"如何"，少用"是不是" |
| **不评判** | 接纳员工的想法和感受 |
| **促承诺** | 引导员工自己承诺行动 |

### 常见误区

| 误区 | 表现 | 避免方法 |
|-----|------|---------|
| **直接给建议** | 告诉员工应该怎么做 | 引导员工自己发现答案 |
| **忽视职业锚** | 推荐与职业锚不匹配的路径 | 先识别职业锚再规划 |
| **目标过大** | 设定不切实际的目标 | 分解为可实现的小目标 |
| **缺乏跟进** | 访谈后无跟进 | 建立定期跟进机制 |
