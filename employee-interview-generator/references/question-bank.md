# 员工访谈问题库

本文件包含 8 种访谈类型的详细问题库，供生成访谈提纲时参考使用。

---

## 📋 问题设计原则

### OPEN 模型

| 字母 | 含义 | 说明 |
|-----|------|-----|
| **O**pen | 开放式 | 避免是/否回答，鼓励详细表达 |
| **P**robe | 追问 | 每个主问题配 2-3 个追问 |
| **E**mpathy | 共情 | 表达理解，建立信任 |
| **N**eutral | 中性 | 避免引导性提问 |

### 问题层次

```
Level 1: 事实性问题（发生了什么）
    ↓
Level 2: 感受性问题（感觉如何）
    ↓
Level 3: 反思性问题（学到什么）
    ↓
Level 4: 行动性问题（接下来怎么做）
```

## 1️⃣ 入职访谈问题库

**适用场景：** 新员工入职 1 周/1 月/3 月跟进

### 适应情况

```
主问题：入职这段时间感觉怎么样？
追问：
- 实际工作内容和面试时了解的一致吗？
- 团队氛围符合你的预期吗？
- 有什么让你惊喜或意外的地方吗？

主问题：工作内容上手还顺利吗？
追问：
- 有没有遇到什么技术或业务上的困难？
- 哪些地方觉得不太顺手？
- 需要什么样的支持？
```

### 团队融入

```
主问题：和团队成员相处得怎么样？
追问：
- 有没有认识一些同事？
- 团队沟通方式适应吗？
- 有什么想了解的团队文化吗？

主问题：和直属上级的沟通还顺畅吗？
追问：
- 上级的管理风格适应吗？
- 反馈机制清楚吗？
- 有什么期望的沟通频率吗？
```

### 期望对齐

```
主问题：你对这个岗位有什么期望？
追问：
- 希望在这个岗位上获得什么成长？
- 短期（3 个月）和长期（1 年）目标是什么？
- 有什么想了解的公司发展信息吗？

主问题：有什么我们可以帮你改善的吗？
追问：
- 工作环境、工具、流程方面？
- 培训或学习资源？
- 其他任何方面？
```

### 试用期目标（如适用）

```
主问题：试用期目标清楚吗？
追问：
- 有什么不清楚的地方吗？
- 觉得目标有挑战性吗？
- 需要什么样的支持来达成？
```

## 2️⃣ 绩效访谈问题库

**适用场景：** 季度/年度绩效评估

### 目标回顾

```
主问题：回顾这个周期，你给自己打几分？为什么？
追问：
- 哪个目标完成得最满意？
- 哪个目标觉得比较遗憾？
- 如果重来一次，会有什么不同做法？

主问题：你觉得最重要的成就是什么？
追问：
- 能具体讲讲你的贡献吗？
- 这个成果对团队/业务有什么价值？
- 过程中遇到了什么挑战？
```

### 差距分析

```
主问题：目标达成过程中遇到的最大挑战是什么？
追问：
- 你是如何应对的？
- 哪些因素影响了你的表现？
- 有什么是外部因素，哪些是内部因素？

主问题：有什么地方觉得可以做得更好吗？
追问：
- 具体是哪些方面？
- 为什么当时没有做到？
- 需要什么支持来改进？
```

### 能力评估

```
主问题：这个周期你提升了哪些能力？
追问：
- 哪项提升最明显？
- 是如何提升的？（项目/培训/自学）
- 还有什么能力想提升？

主问题：你觉得自己的核心优势是什么？
追问：
- 这些优势如何在工作中发挥？
- 有没有被充分使用？
- 如何进一步放大优势？
```

### 发展计划

```
主问题：下阶段你最想提升的能力是什么？
追问：
- 为什么选择这个方向？
- 计划如何提升？
- 需要什么资源或支持？

主问题：你的职业发展目标是什么？
追问：
- 短期（1 年）和长期（3 年）目标？
- 觉得目前和目标的差距在哪里？
- 希望公司如何帮助你？
```

### 工作状态

```
主问题：最近的工作节奏感觉怎么样？
追问：
- 工作和生活的平衡还好吗？
- 压力水平如何？
- 有什么我们可以帮你改善的吗？

主问题：对团队/公司有什么建议吗？
追问：
- 流程、协作、文化方面？
- 有什么应该保持的？
- 有什么应该改进的？
```

## 3️⃣ 离职访谈问题库

**适用场景：** 员工提交离职申请后

### 离职原因探索

```
主问题：能分享一下做出这个决定的主要原因吗？
追问：
- 是什么让你开始考虑离开的？
- 这个决定考虑了多久？
- 有没有什么触发事件？

主问题：除了刚才说的，还有其他原因吗？
追问：
- 这些因素中哪个最关键？
- 有没有什么是我们本可以做但没做的？
- 如果 [关键因素] 能改变，你会考虑留下吗？
```

### 工作体验回顾

```
主问题：在公司工作的这段时间，整体感受如何？
追问：
- 最满意的是什么？
- 最不满意的是什么？
- 有什么遗憾吗？

主问题：对直属上级的管理方式有什么感受？
追问：
- 沟通顺畅吗？
- 得到的支持够吗？
- 有什么建议吗？
```

### 团队反馈

```
主问题：对团队氛围有什么感受？
追问：
- 协作方式怎么样？
- 团队文化适应吗？
- 有什么想说的吗？

主问题：对跨部门协作有什么感受？
追问：
- 有什么阻碍吗？
- 流程顺畅吗？
- 有什么建议？
```

### 改进建议

```
主问题：如果给公司提三个改进建议，会是什么？
追问：
- 这些建议的优先级是怎样的？
- 实施起来有什么难点吗？
- 有什么是我们应该保持的？

主问题：对接班的同事有什么建议吗？
追问：
- 工作交接有什么需要注意的？
- 有什么经验想分享？
- 有什么资源可以推荐？
```

### 未来规划

```
主问题：方便分享一下下一步的计划吗？
追问：
- 是休息还是直接入职新公司？
- 新机会最吸引你的是什么？
- 有什么是我们应该学习的吗？

主问题：以后还愿意保持联系吗？
追问：
- 愿意做公司推荐人吗？
- 有兴趣参与校友活动吗？
- 有考虑过未来回来吗？
```

## 4️⃣ 晋升访谈问题库

**适用场景：** 晋升评审前/后

### 能力证明

```
主问题：你觉得哪些成果证明了你达到了下一职级的要求？
追问：
- 能分享一个最有代表性的项目吗？
- 在这个过程中你展现了哪些核心能力？
- 这些成果的影响力有多大？

主问题：和同职级的同事相比，你的优势是什么？
追问：
- 有什么独特的贡献？
- 如何体现差异化价值？
- 有什么可以量化的成果？
```

### 角色理解

```
主问题：你如何理解下一职级的职责和要求？
追问：
- 和当前职级的主要区别是什么？
- 需要承担哪些新责任？
- 期望的变化是什么？

主问题：晋升后你计划如何开展工作？
追问：
- 工作重点会有什么变化？
- 如何平衡现有工作和新职责？
- 有什么具体计划？
```

### 自我认知

```
主问题：你觉得自己还有哪些需要提升的地方？
追问：
- 和下一职级要求的差距在哪里？
- 计划如何弥补？
- 需要什么支持？

主问题：如果这次没有晋升成功，你会怎么想？
追问：
- 会如何调整？
- 接下来计划怎么做？
- 需要什么样的反馈？
```

### 发展计划

```
主问题：晋升后你的工作重点会是什么？
追问：
- 短期和长期目标分别是什么？
- 如何衡量成功？
- 有什么里程碑？

主问题：需要什么支持来胜任新职级？
追问：
- 培训、导师、资源？
- 团队支持？
- 其他任何方面？
```

## 5️⃣ 职业发展访谈问题库

```
追问：
```

```
追问：
- 如何进一步放大优势？
- 优势如何支持你的目标？
- 有没有被充分使用？
```

### 发展计划

```
主问题：你计划如何提升这些能力？
追问：
- 培训、项目、自学？
- 时间规划是怎样的？
- 如何衡量进展？

主问题：希望公司提供什么支持？
追问：
- 培训资源？

- 导师指导？
- 其他？
```

### 内部机会

```
主问题：有考虑过内部转岗吗？
追问：
- 对哪些团队/方向感兴趣？
- 为什么？
- 有什么顾虑吗？

追问：
- 有什么想改变的？
- 有什么想尝试的？
- 如何更好地发挥价值？
```

## 7️⃣ 挽留访谈问题库

**适用场景：** 核心员工有离职倾向

### 离职意向探索

```
主问题：我听说你最近在考虑一些机会，方便聊聊吗？
追问：
- 是什么让你开始考虑的？
- 考虑了多久？
- 目前进展到什么阶段了？

主问题：能分享一下主要原因吗？
追问：
- 有哪些因素？
- 哪个最关键？
- 有没有什么是我们可以改变的？
```

### 核心诉求

```
主问题：你理想的工作状态是什么样的？
追问：
- 工作内容？
- 发展空间？
- 薪酬福利？
- 其他？

主问题：什么情况下你会考虑留下？
追问：
- 需要什么样的改变？
- 有什么具体条件？
- 优先级是怎样的？
```

### 挽留方案

```
主问题：如果我们能 [满足核心诉求]，你会考虑留下吗？
追问：
- 还需要什么？
- 有什么顾虑吗？
- 需要多长时间考虑？

主问题：你对未来在这边的发展怎么看？
追问：
- 有什么期望？
- 有什么担忧？
- 如何建立信心？
```

## 📌 使用指南

### 问题选择原则

1. **根据访谈目的选择** - 聚焦核心目的，选择相关问题
2. **根据员工情况调整** - 考虑职级、性格、情境
3. **控制问题数量** - 45 分钟访谈选择 8-12 个主问题
4. **预留追问空间** - 每个主问题预留 2-3 个追问

### 问题排序原则

1. **由浅入深** - 先易后难，建立信任
2. **由事实到感受** - 先问事实，再问感受
3. **敏感问题后置** - 建立信任后再问敏感问题
4. **预留缓冲** - 敏感问题前后安排轻松问题

### 追问技巧

```
澄清型：能具体讲讲吗？/ 能举个例子吗？
深入型：为什么？/ 然后呢？
感受型：当时感觉怎么样？
影响型：这对你/团队有什么影响？
```
