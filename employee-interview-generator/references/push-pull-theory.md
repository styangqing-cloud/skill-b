# 离职访谈指南 - 推拉力理论

本指南为使用**推拉力理论**（Push-Pull Theory）进行离职访谈提供详细指导。

---

## 📊 理论介绍

推拉力理论由心理学家 Kurt Lewin 提出，用于解释员工离职决策：

```
┌─────────────────────────────────────────────────────┐
│                  离职决策                            │
│                                                      │
│    推力 (Push)          ←→         拉力 (Pull)      │
│    (现有工作负面)                  (新机会正面)      │
│                                                      │
│  • 工作不满意                    • 更好薪酬          │
│  • 发展受限                      • 更好发展          │
│  • 管理问题                      • 更好文化          │
│  • 薪酬不公                      • 更好平衡          │
│  • 文化不适                      • 新挑战/机会       │
│                                                      │
│              留任力 (Stay) ←→ 移动障碍 (Barrier)     │
│              (留下的理由)          (离开的阻力)      │
│                                                      │
│  • 情感承诺                      • 转换成本          │
│  • 经济依赖                      • 家庭因素          │
│  • 发展机会                      • 地理限制          │
│  • 人际关系                      • 风险规避          │
└─────────────────────────────────────────────────────┘
```

**核心逻辑：**
- **推力 > 拉力 + 留任力 - 移动障碍** → 员工离职
- 离职是四种力量动态平衡的结果

---

## 🎯 四类力量详解

### 1️⃣ 推力（Push Factors）- 现有工作的负面因素

**定义：** 促使员工想要离开当前工作的因素

**常见推力：**
| 推力类型 | 具体表现 | 追问方向 |
|---------|---------|---------|
| **工作不满意** | 工作内容枯燥、缺乏挑战 | "工作内容与期望的差距？" |
| **发展受限** | 晋升无望、学习停滞 | "发展瓶颈是什么？" |
| **管理问题** | 与上级冲突、管理方式不适 | "管理方式有哪些不适？" |
| **薪酬不公** | 薪酬低于市场/内部不公平 | "薪酬期望与现实的差距？" |
| **文化不适** | 价值观冲突、氛围压抑 | "文化哪些方面不适应？" |
| **工作负荷** | 加班过多、压力过大 | "工作负荷对生活的影哬？" |
| **认可缺失** | 贡献不被看见、缺乏反馈 | "哪些贡献被忽视了？" |

**访谈问题示例：**

```
主问题：是什么让你开始考虑离开的？

追问提示（识别推力）：
- 这个想法是从什么时候开始的？当时发生了什么？
- 是单一事件还是多个因素累积的结果？
- 【推力识别】如果用一个词概括，最让你想离开的是什么？

主问题：在现在的工作中，最让你不满意的是什么？

追问提示（识别推力）：
- 这个问题存在多久了？有尝试过解决吗？
- 这个问题对你的日常工作影响有多大？1-10 分打个分
- 【推力识别】如果这个问题能解决，你会考虑留下吗？
```

---

### 2️⃣ 拉力（Pull Factors）- 新机会的正面因素

**定义：** 外部机会对员工的吸引力

**常见拉力：**
| 拉力类型 | 具体表现 | 追问方向 |
|---------|---------|---------|
| **更高薪酬** | 薪资涨幅、股权激励 | "新机会的薪酬吸引力？" |
| **更好发展** | 晋升空间、学习机会 | "新机会的发展空间？" |
| **更好文化** | 更匹配的文化、更开放 | "新公司文化哪里吸引你？" |
| **更好平衡** | 弹性工作、更少加班 | "工作生活平衡的改善？" |
| **新挑战** | 新项目、新领域、创业 | "新机会的挑战性？" |
| **地理位置** | 离家近、城市偏好 | "地理位置的吸引力？" |

**访谈问题示例：**

```
主问题：新机会最吸引你的是什么？

追问提示（识别拉力）：
- 和现在的工作相比，最大的不同是什么？
- 这个机会解决了你现在面临的什么问题？
- 【拉力识别】如果新机会没有这个吸引力，你还会考虑离开吗？

主问题：你理想的工作状态是什么样的？

追问提示（识别拉力）：
- 新机会在哪些方面更接近你的理想状态？
- 现在的工作和理想状态的差距在哪里？
- 【拉力识别】如果现在的工作能达到理想状态，你会留下吗？
```

---

### 3️⃣ 留任力（Stay Factors）- 留下的理由

**定义：** 促使员工考虑留下的因素

**常见留任力：**
| 留任力类型 | 具体表现 | 追问方向 |
|-----------|---------|---------|
| **情感承诺** | 对公司/团队有感情 | "对公司有什么不舍？" |
| **经济依赖** | 薪酬高、福利好 | "现有待遇的吸引力？" |
| **发展机会** | 有晋升/项目机会 | "现有发展机会的吸引力？" |
| **人际关系** | 团队好、上级好 | "团队哪些地方让你不舍？" |
| **工作稳定** | 稳定、风险低 | "稳定性的价值？" |
| **工作意义** | 工作有价值感 | "工作的意义感？" |

**访谈问题示例：**

```
主问题：在现在的工作中，有什么让你不舍的吗？

追问提示（识别留任力）：
- 如果留下来，你最希望保持的是什么？
- 团队/公司有什么是你认可的吗？
- 【留任力识别】如果这些保留，其他条件改善，你会考虑留下吗？

主问题：什么情况下你会考虑留下来？

追问提示（识别留任力）：
- 需要满足什么条件？
- 这些条件中，哪个最重要？
- 【留任力识别】如果公司能满足这些条件，你愿意给多长时间验证？
```

---

### 4️⃣ 移动障碍（Barrier Factors）- 离开的阻力

**定义：** 阻碍员工离职的因素

**常见移动障碍：**
| 障碍类型 | 具体表现 | 追问方向 |
|---------|---------|---------|
| **转换成本** | 损失年假/股权/福利 | "离开有什么损失？" |
| **家庭因素** | 家人反对、照顾责任 | "家人支持你的决定吗？" |
| **地理限制** | 搬家困难、通勤考虑 | "地理位置有限制吗？" |
| **风险规避** | 担心新环境不适应 | "对新机会有什么担忧？" |
| **职业连续性** | 担心中断职业路径 | "对职业连续性的影响？" |
| **声誉顾虑** | 担心被视为"跳槽频繁" | "担心外界看法吗？" |

**访谈问题示例：**

```
主问题：做这个决定有什么顾虑或担忧吗？

追问提示（识别移动障碍）：
- 离开的最大阻力是什么？
- 家人/朋友支持你的决定吗？
- 【障碍识别】如果这个障碍不存在，你的决定会更坚定吗？

主问题：离开现在的工作，你会失去什么？

追问提示（识别移动障碍）：
- 有哪些损失是你考虑过的？
- 这些损失对你来说重要吗？
- 【障碍识别】如果公司能弥补这些损失，会影响你的决定吗？
```

## 🔍 推拉力分析框架

### 力量评估矩阵

```
                    拉力 (新机会吸引力)
                    高              低
                  ┌─────────────┬─────────────┐
                高│  推力主导   │  推力主导   │
                  │  主动离职   │  被动离职   │
     推           │  (高拉力)   │  (低拉力)   │
     力           ├─────────────┼─────────────┤
     (现          │  留任主导   │  犹豫不决   │
     状           │  可能挽留   │  需要分析   │
     不           │  (高拉力)   │  (低拉力)   │
     满           └─────────────┴─────────────┘
     )
```

### 挽留可能性评估

| 推力 | 拉力 | 留任力 | 移动障碍 | 挽留可能性 | 策略 |
|-----|------|-------|---------|-----------|------|
| 高 | 高 | 低 | 低 | 🔴 低 | 尊重决定，保持联系 |
| 高 | 低 | 高 | 高 | 🟡 中 | 解决核心推力 |
| 低 | 高 | 高 | 低 | 🟡 中 | 强化留任力，增加拉力 |
| 低 | 低 | 高 | 高 | 🟢 高 | 可能是冲动，深入沟通 |

## 💬 离职访谈完整提纲

### 一、开场暖场（5 分钟）

**目标：** 建立信任，说明访谈目的和保密原则

**参考话术：**
> "感谢你愿意抽时间聊聊。今天主要是想听听你的真实想法和感受。"
> 
> "你在公司这段时间的表现大家都有目共睹，是非常优秀的。所以公司也很重视你的决定。"
> 
> "今天聊的内容会严格保密，主要是为了帮助公司改进，你放心说就好。"

---

### 二、推力探索（15 分钟）⭐

**主问题 1：** 能分享一下做出这个决定的主要原因吗？

**追问提示：**
- 是什么让你开始考虑离开的？
- 这个决定考虑了多久了？
- 有没有什么触发事件？
- 【推力识别】如果用一个词概括，最让你想离开的是什么？

**主问题 2：** 在现在的工作中，最让你不满意的是什么？

**追问提示：**
- 这个问题存在多久了？有尝试过解决吗？
- 这个问题对你的日常工作影响有多大？1-10 分打个分
- 【推力识别】如果这个问题能解决，你会考虑留下吗？

**主问题 3：** 除了刚才说的，还有其他原因吗？

**追问提示：**
- 这些因素中哪个最关键？
- 如果按重要性排序，前三位是什么？
- 有没有什么是我们本可以做但没做的？

---

### 三、拉力探索（10 分钟）

**主问题 1：** 新机会最吸引你的是什么？

**追问提示：**
- 和现在的工作相比，最大的不同是什么？
- 这个机会解决了你现在面临的什么问题？
- 【拉力识别】如果新机会没有这个吸引力，你还会考虑离开吗？

**主问题 2：** 你理想的工作状态是什么样的？

**追问提示：**
- 新机会在哪些方面更接近你的理想状态？
- 现在的工作和理想状态的差距在哪里？

---

### 四、留任力探索（10 分钟）

**主问题 1：** 在现在的工作中，有什么让你不舍的吗？

**追问提示：**
- 如果留下来，你最希望保持的是什么？
- 团队/公司有什么是你认可的吗？

**主问题 2：** 什么情况下你会考虑留下来？

**追问提示：**
- 需要满足什么条件？
- 这些条件中，哪个最重要？
- 【挽留试探】如果公司能满足这些条件，你愿意给多长时间验证？

---

### 五、移动障碍探索（5 分钟）

**主问题：** 做这个决定有什么顾虑或担忧吗？

**追问提示：**
- 离开的最大阻力是什么？
- 家人/朋友支持你的决定吗？
- 如果这个障碍不存在，你的决定会更坚定吗？

---

### 六、挽留试探（如适用，5 分钟）

**仅在有挽留可能时使用**

**主问题：** 如果我们能 [解决核心推力]，你会考虑留下吗？

**追问提示：**
- 还需要什么条件？
- 有什么顾虑吗？
- 需要多长时间考虑？

---

### 七、改进建议（5 分钟）

**主问题：** 如果给公司提三个改进建议，会是什么？

**追问提示：**
- 这些建议的优先级是怎样的？
- 实施起来有什么难点吗？
- 有什么是我们应该保持的？

---

### 八、总结与结束（5 分钟）

**参考话术：**
> "今天聊了很多，我总结一下关键点... [复述要点]"
> 
> "你提的建议非常有价值，我会整理后反馈给相关部门。"
> 
> "不管最后结果如何，都感谢你这段时间的贡献，也祝你未来发展顺利！"

```markdown
|---------|---------|---------|-----------|
| 发展受限 | 🔴 高 | 晋升通道不清晰 | 是 |
| 薪酬不公 | 🟡 中 | 低于市场 20% | 是 |
| ... | ... | ... | ... |

| 拉力因素 | 吸引力 | 具体表现 | 是否可匹配 |
|---------|-------|---------|-----------|
| 更高薪酬 | 🔴 高 | 涨幅 30% | 部分 |
| 更好发展 | 🟡 中 | 管理岗位 | 是 |
| ... | ... | ... | ... |

## 留任力分析（留下的理由）
| 留任力 | 强度 | 具体表现 |
|-------|-----|---------|
| 团队感情 | 🟡 中 | 与团队合作愉快 |
| 项目兴趣 | 🟢 高 | 负责核心项目 |

## 移动障碍分析（离开的阻力）
| 障碍 | 强度 | 具体表现 |
|-----|-----|---------|
| 股权未归属 | 🟡 中 | 损失 XX 万 |
| 家人支持 | 🟢 高 | 家人支持决定 |

## 挽留可能性评估
**综合评估：** 🟡 中等（40%）

**核心理由：** 
- 推力主要是发展受限，公司有解决空间
- 拉力主要是薪酬，公司可部分匹配
- 留任力中等，员工对团队和项目有感情
- 移动障碍较低，股权损失可协商

**挽留建议：**
1. 明确晋升路径和时间表
2. 薪酬调整至市场 75 分位
3. 增加项目决策权
4. 协商股权保留方案
```

## ⚠️ 注意事项

### 敏感话题处理

| 话题 | 处理建议 |
|-----|---------|
| **薪酬不满** | 倾听并记录，告知会反馈，不当场承诺 |
| **上级矛盾** | 保持中立，了解事实，不站队 |
| **离职倾向坚决** | 尊重决定，收集建议，保持良好关系 |
| **竞对信息** | 不强求透露，尊重隐私 |

### 挽留边界

| 情况 | 挽留策略 |
|-----|---------|
| **推力可解决 + 拉力可匹配** | 积极挽留，制定方案 |
| **推力可解决 + 拉力不可匹配** | 尝试挽留，强调现有优势 |
| **推力不可解决 + 拉力可匹配** | 尊重决定，保持联系 |
| **推力不可解决 + 拉力不可匹配** | 祝福离职，做好交接 |
