# 员工发展访谈指南 - 吉尔伯特 BEM 模型

本指南为使用**吉尔伯特行为工程模型**（Gilbert's Behavior Engineering Model, BEM）进行员工发展访谈提供详细指导。

---

## 📊 理论介绍

吉尔伯特 BEM 模型由 Thomas Gilbert 于 1978 年提出，用于分析影响员工绩效的因素，分为环境因素和个人因素两大类：

```
┌─────────────────────────────────────────────────────┐
│              影响绩效的六大因素                      │
├─────────────────────┬───────────────────────────────┤
│    环境因素 (75%)   │      个人因素 (25%)           │
├─────────────────────┼───────────────────────────────┤
│  1. 信息与反馈      │   4. 知识与技能               │
│  - 目标清晰吗？     │   - 具备所需知识吗？          │
│  - 反馈及时吗？     │   - 具备所需技能吗？          │
├─────────────────────┼───────────────────────────────┤
│  2. 资源与工具      │   5. 天赋与能力               │
│  - 工具到位吗？     │   - 天赋匹配吗？              │
│  - 资源充足吗？     │   - 能力匹配吗？              │
├─────────────────────┼───────────────────────────────┤
│  3. 激励与后果      │   6. 动机与态度               │
│  - 激励有效吗？     │   - 有内在动机吗？            │
│  - 后果明确吗？     │   - 态度积极吗？              │
└─────────────────────┴───────────────────────────────┘
```

**核心逻辑：**
- **环境因素占 75% 影响**：优先优化环境，而非责怪个人
- **个人因素占 25% 影响**：在环境优化后再考虑个人发展
- **从外到内分析**：先环境后个人，避免"培训万能论"

---

## 🎯 六大因素详解

### 1️⃣ 信息与反馈（Information & Feedback）

**定义：** 员工是否清楚目标和获得及时反馈

**诊断问题：**
- 目标是否清晰传达？
- 反馈是否及时有效？
- 员工知道如何改进吗？

**访谈问题示例：**

```
主问题：你清楚自己的工作目标吗？

追问提示：
- 目标是如何传达给你的？
- 目标的优先级清晰吗？
- 当目标变化时，如何通知你？

主问题：你能及时获得工作反馈吗？

追问提示：
- 谁给你反馈？频率如何？
- 反馈对你有帮助吗？
- 你知道如何改进吗？
```

---

### 2️⃣ 资源与工具（Resources & Tools）

**定义：** 员工是否有完成工作所需的资源和工具

**诊断问题：**
- 工具是否支持高效工作？
- 资源是否充足？
- 流程是否顺畅？

**访谈问题示例：**

```
主问题：你有完成工作所需的工具和资源吗？

追问提示：
- 哪些工具对你帮助最大？
- 哪些工具最难用或缺失？
- 申请新工具的流程顺畅吗？

主问题：工作流程顺畅吗？有什么卡点？

追问提示：
- 哪些流程让你觉得效率低？
- 跨部门协作顺畅吗？
- 有什么流程需要优化？
```

---

### 3️⃣ 激励与后果（Incentives & Consequences）

**定义：** 激励是否与绩效对齐，后果是否明确

**诊断问题：**
- 激励是否与贡献匹配？
- 后果（正/负）是否明确？
- 激励是否及时？

**访谈问题示例：**

```
主问题：你觉得付出和回报匹配吗？

追问提示：
- 什么样的行为会被奖励？
- 什么样的行为会被忽视或惩罚？
- 激励及时吗？

主问题：团队的激励机制能激励你更好地工作吗？

追问提示：
- 现有的激励方式有哪些？
- 哪种激励对你最有效？
- 激励和目标达成之间有明确关联吗？
```

---

### 4️⃣ 知识与技能（Knowledge & Skills）

**定义：** 员工是否具备完成工作所需的知识和技能

**诊断问题：**
- 是否具备所需专业知识？
- 是否具备所需技能？
- 培训是否有效？

**访谈问题示例：**

```
主问题：你觉得自己的知识和技能能满足工作要求吗？

追问提示：
- 哪些方面有欠缺？
- 需要什么样的培训？
- 过去的培训对你有帮助吗？

主问题：你希望提升哪些能力？

追问提示：
- 为什么选择这些能力？
- 计划如何提升？
- 需要什么支持？
```

---

### 5️⃣ 天赋与能力（Aptitude & Ability）

**定义：** 员工的先天天赋和基础能力是否匹配岗位

**诊断问题：**
- 天赋与岗位匹配吗？
- 基础能力是否足够？
- 是否需要调整岗位？

**访谈问题示例：**

```
主问题：你觉得自己的工作能发挥优势吗？

追问提示：
- 你的优势是什么？
- 工作如何发挥这些优势？
- 有什么优势没有被充分利用？

主问题：你觉得岗位和你匹配吗？

追问提示：
- 工作内容符合你的兴趣吗？
- 工作强度能承受吗？
- 有没有考虑过调整岗位？
```

---

### 6️⃣ 动机与态度（Motives & Attitudes）

**定义：** 员工的内在动机和工作态度

**诊断问题：**
- 有内在动机吗？
- 工作态度积极吗？
- 什么能激励 TA？

**访谈问题示例：**

```
主问题：什么驱动你努力工作？

追问提示：
- 是成就、认可、成长、还是其他？
- 什么能让你更有动力？
- 什么会让你失去动力？

主问题：你对工作的态度是什么？

追问提示：
- 把工作当任务、职业、还是事业？
- 工作对你来说意味着什么？
- 什么能提升你的工作意义感？
```

## 🔍 绩效问题分析流程

### 步骤 1：明确绩效差距

**问题：**
- 期望绩效是什么？
- 实际绩效是什么？
- 差距在哪里？

### 步骤 2：从外到内分析

按照 BEM 顺序，逐一分析六个因素：
1. 信息与反馈 → 2. 资源与工具 → 3. 激励与后果
4. 知识与技能 → 5. 天赋与能力 → 6. 动机与态度

### 步骤 3：识别根本原因

针对每个因素，问：
- 这个因素是问题原因吗？
- 证据是什么？
- 改善这个因素能解决问题吗？

### 步骤 4：制定改进方案

针对根本原因，制定方案：
- **环境因素**：优化目标、反馈、工具、流程、激励
- **个人因素**：培训、辅导、调岗

## 💬 员工发展访谈完整提纲

### 一、绩效差距分析（15 分钟）

**主问题 1：** 你觉得自己的工作表现怎么样？

**追问提示：**
- 和期望相比，差距在哪里？
- 哪些方面做得好？哪些方面需要改进？
- 什么原因导致了这些差距？

**主问题 2：** 你遇到的最大挑战是什么？

**追问提示：**
- 这个挑战是环境因素还是个人因素？
- 已经尝试过什么解决方法？
- 结果如何？

---

### 二、环境因素分析（25 分钟）⭐

#### 信息与反馈

**主问题：** 你清楚自己的工作目标吗？能获得及时反馈吗？

**追问提示：**
- 目标是如何传达的？清晰吗？
- 反馈频率如何？对你有帮助吗？
- 你知道如何改进吗？

#### 资源与工具

**主问题：** 你有完成工作所需的工具和资源吗？

**追问提示：**
- 哪些工具最难用或缺失？
- 工作流程顺畅吗？有什么卡点？
- 跨部门协作顺畅吗？

#### 激励与后果

**主问题：** 你觉得付出和回报匹配吗？

**追问提示：**
- 什么样的行为会被奖励？
- 激励及时吗？
- 什么能激励你更好地工作？

---

### 三、个人因素分析（15 分钟）

#### 知识与技能

**主问题：** 你觉得自己的知识和技能能满足工作要求吗？

**追问提示：**
- 哪些方面有欠缺？
- 需要什么样的培训？
- 过去的培训对你有帮助吗？

#### 天赋与能力

**主问题：** 你觉得自己的工作能发挥优势吗？

**追问提示：**
- 你的优势是什么？
- 工作如何发挥这些优势？
- 有没有考虑过调整岗位？

#### 动机与态度

**主问题：** 什么驱动你努力工作？

**追问提示：**
- 什么能让你更有动力？
- 什么会让你失去动力？
- 工作对你来说意味着什么？

---

### 四、改进方案制定（15 分钟）

**主问题 1：** 基于刚才的分析，你觉得最需要改进的是什么？

**追问提示：**
- 是环境因素还是个人因素？
- 改进的优先级是什么？
- 你希望公司提供什么支持？

**主问题 2：** 如果让你设计一个改进方案，你会怎么做？

**追问提示：**
- 短期（1-3 个月）做什么？
- 中期（3-6 个月）做什么？
- 长期（6-12 个月）做什么？

---

### 五、总结与承诺（5 分钟）

**参考话术：**
> "我来总结一下今天的讨论... [复述要点]"
> 
> "其中环境方面的改进：[列出环境改进项]"
> 
> "个人发展方面的计划：[列出个人发展项]"
> 
> "公司提供的支持：[列出支持项]"
> 
> "我们约定下次跟进时间：[约定时间]"

## ⚠️ 注意事项

### 分析原则

| 原则 | 说明 |
|-----|------|
| **先环境后个人** | 优先优化环境因素，而非责怪个人 |
| **证据导向** | 基于数据和事实，而非主观判断 |
| **共同参与** | 员工参与分析，而非被动接受 |
| **持续改进** | 定期回顾和调整方案 |

### 常见误区

| 误区 | 表现 | 避免方法 |
|-----|------|---------|
| **培训万能论** | 所有问题都归因于培训不足 | 先分析环境因素 |
| **个人归因** | 忽视环境，责怪个人 | 从外到内系统分析 |
| **一次性解决** | 期望一次方案解决所有问题 | 分阶段持续改进 |
| **忽视动机** | 只关注技能，忽视动机 | 全面分析六个因素 |
