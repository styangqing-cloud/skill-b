## 🎯 各职级访谈策略

### 低职级（P4-P7 / M4-M7 / T4-T7 / S4-S7 / D4-D7）

**访谈重点：** 执行能力、学习成长、工作适应

**核心关注：**
- 任务完成情况
- 技能提升进度
- 团队融入程度
- 工作方法和习惯

**问题设计原则：**
- 具体化：问具体任务和事件
- 引导性：提供清晰的思考框架
- 支持性：关注成长和支持需求

**示例问题：**

```
【绩效访谈】
主问题：这个季度你负责的主要任务有哪些？
追问：
- 哪些任务完成得比较好？
- 遇到了什么困难？如何解决的？
- 需要什么支持来提升？

【职业发展访谈】
主问题：你希望在未来 1 年内提升哪些能力？
追问：
- 为什么选择这些能力？
- 计划如何学习？
- 需要公司提供什么培训？

【离职访谈】
主问题：工作中有什么让你觉得有挑战的？
追问：
- 这些挑战是来自工作本身还是其他方面？
- 有尝试过什么解决方法吗？
- 如果改善这些方面，会影响你的决定吗？
```

---

### 中职级（P8-P9 / M8-M9 / T8-T9 / S8-S9 / D8-D9）

**访谈重点：** 独立负责、项目管理、团队协作

**核心关注：**
- 项目管理和交付
- 跨团队协作
- 方法论沉淀
- 知识分享和传承

**问题设计原则：**
- 项目导向：问项目经验和成果
- 协作视角：问跨团队合作
- 方法论：问经验和方法沉淀

**示例问题：**

```
【绩效访谈】
主问题：这个季度你负责的项目有哪些？成果如何？
追问：
- 项目中你扮演的角色是什么？
- 如何协调资源和推进进度？
- 项目中有什么经验可以沉淀和分享？

【晋升访谈】
主问题：你觉得哪些成果证明了你达到了下一职级的要求？
追问：
- 和当前职级相比，下一职级的核心差异是什么？
- 你如何体现这种差异？
- 还有什么需要提升的地方？

【组织诊断访谈】
主问题：从你的角度看，团队协作有什么可以优化的地方？
追问：
- 跨团队协作的卡点在哪里？
- 有什么流程或机制需要改进？
- 如果让你设计一个改进方案，你会怎么做？
```

---

### 高职级（P10-P11 / M10-M11 / T10-T11 / S10-S11 / D10-D11）

**访谈重点：** 战略规划、组织建设、业务影响

**核心关注：**
- 战略思考和规划
- 组织建设和人才培养
- 业务突破和创新
- 行业洞察和影响力

**问题设计原则：**
- 战略视角：问长期规划和战略思考
- 组织视角：问团队建设和组织设计
- 业务视角：问业务影响和商业价值

**示例问题：**

```
【绩效访谈】
主问题：回顾这一年，你对业务的最大贡献是什么？
追问：
- 这个贡献的战略意义是什么？
- 如何衡量这个贡献的价值？
- 对团队和组织有什么长远影响？

【职业发展访谈】
主问题：你对自己未来 3-5 年的职业规划是什么？
追问：
- 希望在什么领域产生更大影响？
- 需要什么样的平台和机会？
- 公司如何支持你的发展？

【组织诊断访谈】
主问题：从组织层面看，我们面临的最大挑战是什么？
追问：
- 这些挑战的根源是什么？
- 需要什么样的组织变革？
- 你在变革中可以扮演什么角色？

【挽留访谈】
主问题：什么能让你长期留在这里发展？
追问：
- 你希望实现什么样的职业目标？
- 公司如何帮助你实现这些目标？
- 我们需要提供什么样的平台和机会？
```

### 专家级（P12+ / M12+ / T12+ / S12+ / D12+）

**访谈重点：** 行业影响、创新突破、人才培养

**核心关注：**
- 行业洞察和趋势判断
- 技术创新和业务突破
- 高端人才培养和梯队建设
- 生态建设和战略合作

**问题设计原则：**
- 行业视角：问行业趋势和竞争格局
- 创新视角：问技术/业务创新
- 生态视角：问生态建设和战略合作
- 传承视角：问人才培养和知识传承

**示例问题：**

```
【绩效访谈】
主问题：这一年你在行业/技术领域有什么突破性贡献？
追问：
- 这个突破对行业和公司的意义是什么？
- 如何建立持续创新的机制？
- 对团队和公司能力建设有什么贡献？

【职业发展访谈】
主问题：你希望在未来实现什么样的行业影响力？
追问：
- 需要什么样的平台和资源支持？
- 如何平衡个人影响力和组织建设？
- 公司如何与你共同成长？

【组织诊断访谈】
主问题：从行业视角看，我们的组织能力和竞争对手相比如何？
追问：
- 我们的核心优势是什么？如何保持？
- 我们的短板在哪里？如何补齐？
- 需要什么样的组织变革来支撑战略？

【挽留访谈】
主问题：什么能让你在这里长期投入并实现你的愿景？
追问：
- 你的愿景是什么？
- 公司如何支持你实现这个愿景？
- 我们需要建立什么样的合作模式？
```

## 📋 职级对应的六盒模型维度选择

### 低职级（P4-P7 / T4-T7）

**优先维度：** 支持机制、关系、奖励

**诊断重点：**
- 工具和资源是否到位
- 团队融入和协作
- 激励机制是否有效

### 中职级（P8-P9 / T8-T9）

**优先维度：** 结构、关系、支持机制

**诊断重点：**
- 职责分工和流程
- 跨团队协作
- 项目管理机制

### 高职级（P10-P11 / T10-T11）

**优先维度：** 目的、结构、领导

**诊断重点：**
- 战略方向和目标对齐
- 组织架构设计
- 领导力和决策机制

### 专家级（P12+ / T12+）

**优先维度：** 目的、领导、支持机制

**诊断重点：**
- 战略愿景和行业定位
- 创新机制和资源支持
- 高端人才培养

## 📋 职级对应的职业锚识别

### 低职级（P4-P7 / T4-T7）

**常见职业锚：**
- 技术/职能型（追求技能提升）
- 安全/稳定型（追求稳定发展）
- 生活方式型（追求工作生活平衡）

**识别问题：**
```
- 你理想的工作状态是什么样的？
- 什么能让你感到安心和稳定？
- 工作在你生活中的位置是什么？
```

### 中职级（P8-P9 / T8-T9）

**常见职业锚：**
- 技术/职能型（追求专业深度）
- 管理型（追求管理责任）
- 自主/独立型（追求自主权）

**识别问题：**
```
- 你更喜欢专业深耕还是管理他人？
- 你希望有多大的自主权？
- 什么能让你感到成就感？
```

### 高职级（P10-P11 / T10-T11）

**常见职业锚：**
- 管理型（追求领导责任）
- 创业/创造型（追求创新）
- 服务/奉献型（追求社会价值）

**识别问题：**
```
- 你希望领导多大的组织？
- 你希望创造什么样的价值？
- 什么能激励你持续投入？
```

### 专家级（P12+ / T12+）

**常见职业锚：**
- 创业/创造型（追求行业变革）
- 服务/奉献型（追求社会影响）
- 纯挑战型（追求行业难题）

**识别问题：**
```
- 你希望在行业留下什么影响？
- 什么挑战能让你保持激情？
- 你如何定义成功？
```

## ⚠️ 注意事项

### 职级识别

| 情况 | 处理方式 |
|-----|---------|
| 职级信息缺失 | 询问 HR 或从文件/链接中提取 |
| 跨序列转岗 | 同时参考原序列和目标序列（P/M/T/S/D） |
| 职级体系不同 | 映射到对应级别（低/中/高/专家） |

### 问题调整

| 因素 | 调整方向 |
|-----|---------|
| 职级低但表现优秀 | 适当提升问题深度 |
| 职级高但新任 | 增加支持机制相关问题 |
| 跨级别晋升 | 关注角色转换和适应 |

### 敏感处理

| 情况 | 处理建议 |
|-----|---------|
| 职级与能力不匹配 | 避免直接评价，关注发展 |
| 晋升失败 | 关注感受和后续计划 |
| 降职/转岗 | 关注原因和适应情况 |
