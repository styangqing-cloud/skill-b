# 离职访谈指南 - 推拉力理论

本指南为使用推拉力理论进行离职访谈提供详细指导。

---

## 理论介绍

推拉力理论由心理学家 Kurt Lewin 提出,用于解释员工离职决策:

```
┌─────────────────────────────────────────────────────┐
│                  离职决策                            │
│                                                      │
│    推力 (Push)          ←→         拉力 (Pull)      │
│    (现有工作负面)                  (新机会正面)      │
│                                                      │
│  • 工作不满意                    • 更好薪酬          │
│  • 发展受限                      • 更好发展          │
│  • 管理问题                      • 更好文化          │
│  • 薪酬不公                      • 更好平衡          │
│  • 文化不适                      • 新挑战/机会       │
│                                                      │
│              留任力 (Stay) ←→ 移动障碍 (Barrier)     │
│              (留下的理由)          (离开的阻力)      │
│                                                      │
│  • 情感承诺                      • 转换成本          │
│  • 经济依赖                      • 家庭因素          │
│  • 发展机会                      • 地理限制          │
│  • 人际关系                      • 风险规避          │
└─────────────────────────────────────────────────────┘
```

**核心逻辑**:
- **推力 > 拉力 + 留任力 - 移动障碍** → 员工离职
- 离职是四种力量动态平衡的结果

---

## 四类力量详解

### 1. 推力（Push Factors）- 现有工作的负面因素

**定义**: 促使员工想要离开当前工作的因素

**常见推力**:
| 推力类型 | 具体表现 | 追问方向 |
|---------|---------|---------|
| **工作不满意** | 工作内容枯燥、缺乏挑战 | "工作内容与期望的差距？" |
| **发展受限** | 晋升无望、学习停滞 | "发展瓶颈是什么？" |
| **管理问题** | 与上级冲突、管理方式不适 | "管理方式有哪些不适？" |
| **薪酬不公** | 薪酬低于市场/内部不公平 | "薪酬期望与现实的差距？" |
| **文化不适** | 价值观冲突、氛围压抑 | "文化哪些方面不适应？" |
| **工作负荷** | 加班过多、压力过大 | "工作负荷对生活的影响？" |
| **认可缺失** | 贡献不被看见、缺乏反馈 | "哪些贡献被忽视了？" |

**访谈问题示例**:
- "是什么让你开始考虑离开的？"
- "在现在的工作中,最让你不满意的是什么？"
- "如果用一个词概括,最让你想离开的是什么？"

### 2. 拉力（Pull Factors）- 新机会的正面因素

**定义**: 外部机会对员工的吸引力

**常见拉力**:
| 拉力类型 | 具体表现 | 追问方向 |
|---------|---------|---------|
| **更高薪酬** | 薪资涨幅、股权激励 | "新机会的薪酬吸引力？" |
| **更好发展** | 晋升空间、学习机会 | "新机会的发展空间？" |
| **更好文化** | 更匹配的文化、更开放 | "新公司文化哪里吸引你？" |
| **更好平衡** | 弹性工作、更少加班 | "工作生活平衡的改善？" |
| **新挑战** | 新项目、新领域、创业 | "新机会的挑战性？" |
| **地理位置** | 离家近、城市偏好 | "地理位置的吸引力？" |

**访谈问题示例**:
- "新机会最吸引你的是什么？"
- "和现在的工作相比,最大的不同是什么？"
- "如果新机会没有这个吸引力,你还会考虑离开吗？"

### 3. 留任力（Stay Factors）- 留下的理由

**定义**: 促使员工考虑留下的因素

**常见留任力**:
| 留任力类型 | 具体表现 | 追问方向 |
|-----------|---------|---------|
| **情感承诺** | 对公司/团队有感情 | "对公司有什么不舍？" |
| **经济依赖** | 薪酬高、福利好 | "现有待遇的吸引力？" |
| **发展机会** | 有晋升/项目机会 | "现有发展机会的吸引力？" |
| **人际关系** | 团队好、上级好 | "团队哪些地方让你不舍？" |
| **工作稳定** | 稳定、风险低 | "稳定性的价值？" |
| **工作意义** | 工作有价值感 | "工作的意义感？" |

**访谈问题示例**:
- "在现在的工作中,有什么让你不舍的吗？"
- "如果留下来,你最希望保持的是什么？"
- "如果公司能满足这些条件,你愿意给多长时间验证？"

### 4. 移动障碍（Barrier Factors）- 离开的阻力

**定义**: 阻碍员工离职的因素

**常见移动障碍**:
| 障碍类型 | 具体表现 | 追问方向 |
|---------|---------|---------|
| **转换成本** | 损失年假/股权/福利 | "离开有什么损失？" |
| **家庭因素** | 家人反对、照顾责任 | "家人支持你的决定吗？" |
| **地理限制** | 搬家困难、通勤考虑 | "地理位置有限制吗？" |
| **风险规避** | 担心新环境不适应 | "对新机会有什么担忧？" |
| **职业连续性** | 担心中断职业路径 | "对职业连续性的影响？" |
| **声誉顾虑** | 担心被视为"跳槽频繁" | "担心外界看法吗？" |

**访谈问题示例**:
- "做这个决定有什么顾虑或担忧吗？"
- "离开的最大阻力是什么？"
- "如果这个障碍不存在,你的决定会更坚定吗？"

## 离职访谈提纲

### 一、开场暖场（5 分钟）
- 建立信任
- 说明目的和保密原则

### 二、推力探索（15 分钟）⭐
- 主要决定因素
- 触发事件
- 最不满意的地方

### 三、拉力探索（10 分钟）
- 新机会的吸引力
- 理想工作状态

### 四、留任力探索（10 分钟）
- 不舍的地方
- 留下的条件

### 五、移动障碍探索（5 分钟）
- 顾虑和担忧
- 离开的阻力

### 六、挽留试探（如适用,5 分钟）
- 核心诉求
- 挽留方案

### 七、改进建议（5 分钟）
- 给公司的建议
- 应该保持的

### 八、总结与结束（5 分钟）
- 复述要点
- 祝福未来

## 挽留可能性评估

| 推力 | 拉力 | 留任力 | 移动障碍 | 挽留可能性 | 策略 |
|-----|------|-------|---------|-----------|------|
| 高 | 高 | 低 | 低 | 🔴 低 | 尊重决定,保持联系 |
| 高 | 低 | 高 | 高 | 🟡 中 | 解决核心推力 |
| 低 | 高 | 高 | 低 | 🟡 中 | 强化留任力,增加拉力 |
| 低 | 低 | 高 | 高 | 🟢 高 | 可能是冲动,深入沟通 |

## 注意事项

### 敏感话题处理

| 话题 | 处理建议 |
|-----|---------|
| **薪酬不满** | 倾听并记录,告知会反馈,不当场承诺 |
| **上级矛盾** | 保持中立,了解事实,不站队 |
| **离职倾向坚决** | 尊重决定,收集建议,保持良好关系 |
| **竞对信息** | 不强求透露,尊重隐私 |

### 挽留边界

| 情况 | 挽留策略 |
|-----|---------|
| **推力可解决 + 拉力可匹配** | 积极挽留,制定方案 |
| **推力可解决 + 拉力不可匹配** | 尝试挽留,强调现有优势 |
| **推力不可解决 + 拉力可匹配** | 尊重决定,保持联系 |
| **推力不可解决 + 拉力不可匹配** | 祝福离职,做好交接 |
