# 员工发展访谈指南 - 吉尔伯特 BEM 模型

本指南为使用吉尔伯特行为工程模型进行员工发展访谈提供指导。

---

## 核心理论:吉尔伯特行为工程模型 (BEM)

该模型由 Thomas Gilbert 于 1978 年提出,将影响因素分为两大类共六点,核心观点是**优先优化环境,而非责怪个人**。

### 影响绩效的六大因素结构

| 类别 | 占比 | 因素 | 关键问题 |
| :--- | :--- | :--- | :--- |
| **环境因素** (Environment) | **75%** | **1. 信息与反馈** | 目标清晰吗？反馈及时吗？ |
| | | **2. 资源与工具** | 工具到位吗？资源充足吗？ |
| | | **3. 激励与后果** | 激励有效吗？后果明确吗？ |
| **个人因素** (Individual) | **25%** | **4. 知识与技能** | 具备所需知识吗？具备所需技能吗？ |
| | | **5. 天赋与能力** | 天赋匹配吗？能力匹配吗？ |
| | | **6. 动机与态度** | 有内在动机吗？态度积极吗？ |

---

## 关键分析原则

1. **优先级顺序**: 遵循"从外到内"的分析顺序（因素 1 → 6）
2. **核心逻辑**:
   - **环境因素占 75% 影响**: 应优先排查环境支持是否到位
   - **个人因素占 25% 影响**: 在确保环境无障碍后,再考虑个人能力提升或岗位匹配

---

## 员工发展访谈提纲

### 一、绩效差距分析 (15分钟)
- 聚焦期望绩效与实际绩效的差距
- 探讨遇到的最大挑战及其原因

### 二、环境因素分析 (25分钟) ⭐ (重点)

**信息与反馈**
- 目标传达是否清晰？反馈是否有帮助？

**资源与工具**
- 哪些工具缺失或难用？流程卡点在哪里？

**激励与后果**
- 付出与回报是否匹配？激励机制是否有效？

### 三、个人因素分析 (15分钟)

**知识与技能**
- 当前技能是否满足要求？需要什么培训？

**天赋与能力**
- 优势是否得到发挥？岗位是否匹配？

**动机与态度**
- 工作的驱动力是什么？工作意义感如何？

### 四、改进方案制定 (15分钟)
- 确定最需要改进的领域及优先级
- 设计短期（1-3月）、中期（3-6月）、长期（6-12月）行动计划

### 五、总结与承诺 (5分钟)
- 复述讨论要点,明确环境改进项与个人发展项
- 约定下次跟进时间

## 注意事项

### 常见误区
- 避免"个人归因":优先考虑环境因素
- 避免"一次性解决":持续跟进
- 避免"培训万能论":培训不能解决环境问题

### 访谈技巧
- 基于证据,而非主观判断
- 持续跟进,而非一次访谈
- 关注系统性问题,而非个人短板
