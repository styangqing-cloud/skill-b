# 腾讯文档集成指南

本技能支持与腾讯文档技能集成，可将生成的访谈提纲直接创建为腾讯文档。

---

## 🔧 前置条件

1. 已安装腾讯文档技能：`/root/.openclaw/workspace/skills/tencent-docs/`
2. 已配置环境变量：`TENCENT_DOCS_TOKEN`
3. 已验证 mcporter 配置：`mcporter list | grep tencent-docs`

---

## 📋 使用方式

### 方式 1：生成后手动创建

```
1. 使用本技能生成访谈提纲（Markdown 格式）
2. 复制生成的 Markdown 内容
3. 调用腾讯文档技能创建文档：
   mcporter call tencent-docs.create_smartcanvas_by_markdown
```

### 方式 2：自动创建（推荐）

当用户表示需要腾讯文档时，自动调用腾讯文档技能：

```
用户：帮我生成一个绩效访谈提纲，顺便创建到腾讯文档

你：
1. 收集必要信息（员工信息、访谈类型、目的）
2. 生成访谈提纲
3. 调用腾讯文档技能创建文档
4. 返回文档链接
```

---

## 🛠️ 调用示例

### 创建智能文档

```bash
mcporter call tencent-docs.create_smartcanvas_by_markdown --args '{
  "title": "员工访谈提纲 - 绩效面谈 - 张 XXX",
  "markdown": "# 员工访谈提纲\n\n## 📌 访谈基本信息\n\n- 员工姓名：张 XXX\n- 职位/职级：后端开发工程师 P6\n- 所属部门：技术部 - 基础架构组\n- 入职时间：2023 年 3 月\n- 访谈类型：绩效访谈\n- 预计时长：45-60 分钟\n- 访谈人：HR 李 XXX\n- 访谈日期：2025 年 Q1 绩效周期\n\n## 🎯 访谈目的\n\n1. 回顾 Q4 绩效目标达成情况\n2. 识别工作中的挑战和改进空间\n3. 讨论 2025 年个人发展计划\n4. 了解资源和支持需求\n\n... [更多内容]",
  "parent_id": "你的空间 ID（可选）"
}'
```

### 响应格式

```json
{
  "file_id": "YequjdOnoMLC",
  "url": "https://docs.qq.com/aio/DWWVxdWpkT25vTUxD"
}
```

## 🔄 工作流示例

### 完整流程

```
1. 用户请求：帮我生成一个绩效访谈提纲，创建到腾讯文档

2. 信息收集：
   - 员工信息：姓名、职位、部门、入职时间
   - 访谈类型：绩效访谈
   - 访谈目的：回顾绩效、讨论发展

3. 生成提纲：
   - 根据信息生成结构化访谈提纲

4. 创建文档：
   - 调用腾讯文档技能
   - 传入提纲 Markdown 内容
   - 获取文档链接

5. 返回结果：
   - 访谈提纲已生成
   - 腾讯文档链接：https://docs.qq.com/aio/XXX
   - 提醒用户可以进行编辑和记录
```

---

## ⚠️ 注意事项

### 内容格式
- 腾讯文档智能文档支持 Markdown 语法
- 表格、列表、标题等格式会自动渲染
- 特殊字符无需转义

### 权限管理
- 创建的文档默认仅自己可见
- 可通过腾讯文档分享功能分享给他人
- 建议设置合适的查看/编辑权限

### 文档更新
- 访谈后可直接在腾讯文档中补充记录
- 支持多人协作编辑
- 历史记录自动保存
