## ✨ 新增功能

### 1. 招聘面试支持

**支持的面试类型：**
- ✅ 简历筛选后（初面/业务面）
- ✅ 终面前（高管面/HR 面）
- ✅ 校招面试（批量面试）
- ✅ 社招面试（全年）
- ✅ 内推面试

**输入信息要求：**
```yaml
候选人信息:
  - 基本信息（姓名/公司/职位/年限）
  - 教育背景（学校/专业/学历）
  - 工作经历（公司/职责/成就）
  - 项目经验（角色/成果）
  - 技能清单
  - 跳槽动机

应聘岗位:
  - 岗位 JD（职责/要求）
  - 岗位级别（P5/P6/P7...）
  - 胜任力模型

面试信息:
  - 面试轮次（初面/业务面/高管面）
  - 面试官角色
  - 面试时长/形式
```

---

### 2. STAR 面试法框架 ⭐

**问题设计原则：**
- **(S) 情境**：当时的背景和情况
- **(T) 任务**：你的目标和任务
- **(A) 行动**：你采取的具体行动
- **(R) 结果**：最终成果和影响

**示例问题：**
```markdown
Q. AI 产品从 0 到 1 经验 ⭐⭐⭐

主问题：你提到主导了 AI 助手产品从 0 到 1，请详细描述这个过程。

STAR 追问：
- (S) 情境：市场背景？用户痛点？竞品情况？
- (T) 任务：产品定位？核心目标用户？
- (A) 行动：MVP 功能确定？优先级决策？关键挑战？
- (R) 结果：DAU 50 万用时？留存率 30% 计算方式？

考察点：产品思维、用户洞察、决策能力
评分标准：5 星评级
```

---

### 3. 面试评估表（100 分制）⭐

**评估维度：**
```yaml
能力匹配 (40%):
  - AI 产品专业能力：15%
  - 技术理解力：10%
  - 数据分析能力：10%
  - 产品迭代方法论：5%

经验匹配 (30%):
  - 相关行业经验：10%
  - 项目管理经验：10%
  - 从 0 到 1 经验：10%

文化匹配 (20%):
  - 价值观认同：8%
  - 工作风格匹配：6%
  - 沟通协作：6%

潜力评估 (10%):
  - 学习能力：5%
  - 成长意愿：5%

总分：___ / 100 分

推荐等级:
- 强烈推荐（≥90 分）
- 推荐（75-89 分）
- 待定（60-74 分）
- 不推荐（<60 分）
```

---

### 4. 红线信号检查清单 ⭐

**能力真实性验证：**
- [ ] 项目成果是否有具体数据支撑？
- [ ] 个人贡献是否清晰（主导 vs 参与）？
- [ ] 技术理解是否深入（而非表面术语）？

**动机匹配度：**
- [ ] 跳槽原因是否合理？
- [ ] 对岗位的理解是否准确？
- [ ] 职业规划是否与公司发展一致？

**文化匹配度：**
- [ ] 工作风格是否与团队匹配？
- [ ] 价值观是否认同？
- [ ] 沟通方式是否顺畅？

